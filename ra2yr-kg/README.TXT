Red Alert 2 + Yuri's Revenge (with keygen)
+==HOW TO INSTALL==+
1) Extract files from .ZIP Archive
2) Run Setup/RA2YR-Setup.exe
3) IMPORTANT! Install to: ** C:\Westwood\RA2 **
4) Copy all .MIX files in the OGSOURCE folder to: ** C:\Westwood\RA2 **

5) Go to https://github.com/FunkyFr3sh/cnc-ddraw/releases
6) Find the version for your OS:
   -> cnc-ddraw.zip for newer systems (XP and up)
   -> cnc-ddraw_win2000_ia32_msvc.zip for Windows 2000 (32bit)
7) Download the file
8) Extract to: ** C:\Westwood\RA2 **
9) Edit ddraw.ini in ** C:\Westwood\RA2 **
   9b. MOST IMPORTANT: Set screen resolution (width= and height=)
